# 채용 공고 회사 이메일 수집기

사람인(Saramin) + 잡코리아(JobKorea)에서 경영지원 관련 구인 공고를 검색하고, 해당 회사의 **공식 홈페이지에 공개된 이메일**을 자동 수집하는 도구입니다.

## 수집 흐름

```
사람인/잡코리아 공고 검색 → 회사명 추출 → 기업 페이지에서 대표자명/홈페이지 수집
→ 회사 공식 사이트에서 공개 이메일 수집 → 50건 채우면 자동 종료 → CSV 출력
```

## 주요 기능

- **2개 사이트** 동시 검색: 사람인 + 잡코리아
- **8개 키워드**: 경영지원, 경리, 총무, 비서, 과제비 관리, 정부지원사업 관리, 연구비 관리, 인사
- **최근 1주일** 공고만 필터링
- **목표 기반 수집**: 이메일 50건 모이면 자동 종료 (부족 시 다음 페이지 자동 탐색)
- **깊이 탐색**: 메인 페이지, contact/about, 문의하기 폼, footer 영역까지 이메일 탐색
- **팀 공유 중복 체크**: `output/` CSV를 Git으로 공유하여 팀원 간 중복 수집 방지
- **회사 쿨다운**: 같은 회사가 CSV에 다시 포함되려면 30일 필요
- **쓰레기 필터링**: 호스팅 이메일, sentry, placeholder 등 자동 제외

## 설치

```bash
git clone https://github.com/dlwngml0721/saramin-scraper.git
cd saramin-scraper
pip install -r requirements.txt
```

## 사용법

### 혼자 사용할 때

```bash
python scraper.py
```

### 팀으로 사용할 때

중복 수집을 방지하려면 아래 순서를 반드시 지켜주세요:

```bash
# 1. 최신 이력 받기 (다른 사람이 수집한 CSV 반영)
git pull

# 2. 수집 실행
python scraper.py

# 3. 결과 공유
git add output/
git commit -m "수집 결과 추가"
git push
```

**주의:** 두 사람이 동시에 실행하면 같은 회사를 수집할 수 있습니다.
실행 전에 항상 `git pull`을 먼저 하고, 동시 실행만 피하면 됩니다.

## 출력

- CSV 파일: `output/recruit_YYYYMMDD_HHMM.csv`
- 컬럼: `기업명`, `대표자명`, `공고 키워드`, `이메일`, `홈페이지`, `출처`

## 중복 관리

| 구분 | 방식 | 공유 범위 |
|------|------|-----------|
| CSV 이력 | `output/*.csv`를 읽어서 이미 수집된 회사 스킵 | Git으로 팀 전체 공유 |
| 로컬 DB | `scraper_history.db`에 공고 ID 캐시 | 개인 로컬 전용 (.gitignore) |

- 같은 회사가 CSV에 다시 포함되려면 **30일** 쿨다운 필요
- DB가 없어도 CSV만으로 중복 체크 동작

## 설정 변경

`scraper.py` 상단에서 수정 가능:

```python
# 검색 키워드
KEYWORDS = ["경영지원", "경리", "총무", ...]

# 이메일 수집 목표 건수 (이 수만큼 모이면 자동 종료)
TARGET_EMAILS = 50

# 같은 회사 재포함 간격 (일)
COMPANY_COOLDOWN_DAYS = 30

# 키워드당 최대 검색 페이지 (공고가 부족할 때 더 깊이 탐색)
MAX_PAGES = 10
```

## 주의사항

- 사람인/잡코리아 이용약관에 따라 과도한 크롤링은 제한될 수 있습니다
- 수집된 이메일로 광고성 메일 발송 시 정보통신망법의 수신거부(opt-out) 요건을 준수해야 합니다
- 회사 공식 홈페이지에 공개된 이메일만 수집하며, 개인 이메일은 수집하지 않습니다
