# 사람인 경영지원 공고 회사 이메일 수집기

사람인(Saramin)에서 경영지원 관련 구인 공고를 검색하고, 해당 회사의 **공식 홈페이지에 공개된 이메일**을 자동 수집하는 도구입니다.

## 수집 흐름

```
사람인 공고 검색 → 회사명 추출 → 사람인 기업 페이지에서 대표자명/홈페이지 수집
→ 회사 공식 사이트에서 공개 이메일 수집 → CSV 출력
```

## 주요 기능

- **8개 키워드** 동시 검색: 경영지원, 경리, 총무, 비서, 과제비 관리, 정부지원사업 관리, 연구비 관리, 인사
- **최근 1주일** 공고만 필터링
- **깊이 탐색**: 메인 페이지, contact/about 등 고정 경로, 문의하기 폼 링크, footer 영역까지 이메일 탐색
- **중복 제거**: SQLite DB에 수집 이력 저장, 재실행 시 이전에 수집한 회사 자동 스킵
- **쓰레기 필터링**: 호스팅 이메일, sentry, placeholder 등 자동 제외
- **이메일 없는 회사 제외**: CSV에는 이메일 수집 성공한 회사만 포함

## 설치 & 실행

```bash
# 1. 클론
git clone <repo-url>
cd saramin_scraper

# 2. 의존성 설치
pip install -r requirements.txt

# 3. 실행
python scraper.py
```

## 출력

- CSV 파일: `output/saramin_YYYYMMDD_HHMM.csv`
- 컬럼: `기업명`, `대표자명`, `공고 키워드`, `이메일`, `홈페이지`

## 재실행 시

`scraper_history.db`에 이전 수집 이력이 남아 있어, 이미 수집한 회사는 자동으로 건너뜁니다.
DB를 초기화하고 싶으면:

```bash
rm scraper_history.db
```

## 키워드 변경

`scraper.py` 상단의 `KEYWORDS` 리스트를 수정하면 됩니다:

```python
KEYWORDS = [
    "경영지원", "경리", "총무", "비서",
    "과제비 관리", "정부지원사업 관리", "연구비 관리", "인사"
]
```

## 주의사항

- 사람인 이용약관에 따라 과도한 크롤링은 제한될 수 있습니다
- 수집된 이메일로 광고성 메일 발송 시 정보통신망법의 수신거부(opt-out) 요건을 준수해야 합니다
- 회사 공식 홈페이지에 공개된 이메일만 수집하며, 개인 이메일은 수집하지 않습니다
